# Phase 2: Navigation Shell

## Goal
Set up Expo Router tab navigation with empty screens — the skeleton the user taps through.

## What you'll learn
- How Expo Router uses the file system to define routes
- How tab navigation works in React Native vs web routing
- How layouts nest and share UI

## Tasks
- [ ] Create tab layout (Home, History, Profile)
- [ ] Create empty screen files for all routes (architecture.md navigation map)
- [ ] Configure tab bar labels and icons
- [ ] Set up nested routes (template editor, session detail, etc.)
- [ ] Test navigation flow end-to-end

## Notes
- Web runs in shell/demo mode — all routes should render, but no local persistence is expected.
